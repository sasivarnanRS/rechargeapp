# RechargeApp

A mobile recharge platform with instant recharges and secure payments.

## Features
- Instant mobile recharges
- Secure payment gateway integration
- Plan exploration by operator
- Recharge history tracking
- Admin dashboard for plan management

## Tech Stack
- Frontend: React (Vite), Tailwind CSS, Framer Motion
- Backend: Node.js, Express, MongoDB
- Authentication: JWT

## Current Status
- Frontend flow updated: About page as landing, public Plans page.
- Login required for recharge actions.
